## Patitas Felices Repositorio
Proyecto del equipo Patitas Felices — gestión y desarrollo colaborativo

## Estructura del repositorio
- /src       → Código fuente
- /docs      → Documentación
- /tests     → Tests
- /assets    → Recursos estáticos

## Ramas
- main       → Producción (protegida)
- develop    → Desarrollo diario
- feature/*  → Funcionalidades

## Integraciones
- Microsoft Teams: notificaciones y colaboración
- Trello: seguimiento de historias de usuario (vínculos desde tarjetas)
